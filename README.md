# PEMUDA - MILENIAL SHOP (BACKEND)
  This is a platform to control you e-commerce frontend. this app building with laravel 7x
 
 
 ## Features
   - Product Module
   - Transaction Module
   - E-Commerce Settings
   - Landing Page Settings
  
 ## Usage (Before Deploy)
  - Before deploy you should changes env 
  - run a migrate 
  ```bash
  php artisan migrate
  ```
  - run a database seed
  ```bash 
  php artisan db:seed
  ```
  
  now you can use the backend
